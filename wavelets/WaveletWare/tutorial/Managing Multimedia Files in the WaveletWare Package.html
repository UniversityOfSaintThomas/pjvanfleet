Importing Multimedia Files

The WaveletWare package is designed for use with image, audio and data files. The package comes with a prescribed set of these files but users can certainly add their own files to the package. Deleting stock files is strongly discouraged as doing so will corrupt some elements of the documentation.

This tutorial deals with managing multimedia files. In particular, it will discuss the directory structure for such files, how to add files to the package, and how to import these files.

The WaveletWare package comes with a few functions that assist users with directory structure and acceptable formats for multimedia files.

AudioInfo[opts]Information about package audio files
DataFormatsAcceptable Mathematica data files
DataInfo[opts]Information about package data files
ImageFormatsAcceptable Mathematica image files
ImageInfo[opts]Information about package image files.

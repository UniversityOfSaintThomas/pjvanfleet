BWT


returns the biorthogonal wavelet transform of a vector, a matrix or a list of three matrices of equal dimension.

DetailsDetails

  • The following options can be given:
  • NumIterations1NumIterations is an option used by several functions in the WaveletWare package. It indicates the number of iterations of the wavelet transform performed on the data.
    ComputationNumericalComputation is an option for wavelet transform functions or the CE function that indicates whether the comptuation should be done numerically or symbolically.
    BoundaryFalseBoundary iis an option for several routines that either compute or use the biorthogonal wavelet transform. If set to True, a symmetric biorthogonal transformation is computed.
  • For vectors of length n, the output data is returned as a vector in the form (, , , ..., ) where the length of any component with subscript j is n/2j.
  • For a matrix (or matrices) of dimension n x m, the output data is returned as a matrix (or matrices) where the highpass portions of iteration 1 are placed in the upper right (vertical), lower left (horizontal), and lower right (diagonal) spots of the matrix, and subsequent iterations (or the lowpass portion, if NumIterations is 1), are placed in a similar manner in the upper left portion of the matrix.
  • A second option for Computation is Symbolic.
  • Symbolic computation will be overridden if either the input filter or the data are given in numerical form.
  • Boundary can be set to True to exploit the symmetric nature of a biorthogonal filter pair and provide a more accurate representation of the transformed data at the beginning and end of data sets.

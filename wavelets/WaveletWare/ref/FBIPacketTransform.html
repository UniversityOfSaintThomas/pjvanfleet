FBIPacketTransform

FBIPacketTransform[data]
computes the wavelet packet transform or a matrix or list of three matrices using the FBI best basis tree.

DetailsDetails

  • FBIPacketTransform[data] is equivalent to WPT[data,CDF97[],NumIterations->5,FixedTree->FBITree[],CompositeTree->True,Computation->Numerical,Boundary->True]
  • The input data must be a matrix or a list of three matrices of equal dimension.
  • Unlike WPT, FBIPacketTransform returns only the packet transform (as opposed to WPT which returns the packet transform and the best basis tree(s)).

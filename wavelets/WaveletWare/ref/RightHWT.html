RightHWT


computes a, where is a discrete wavelet transform matrix associated with the Haar filter and a is a matrix or a list of three matrices each of the same dimensions. The column dimension must be even.

DetailsDetails

  • The following options can be given:
  • Shift0Shift is an option for discrete wavelet transform functions to indicate if the lowpass/highpass filters should be shifted for the computation.
    ComputationNumericalComputation is an option for wavelet transform functions or the CE function that indicates whether the comptuation should be done numerically or symbolically.
    OrthogonalFalseOrthogonal is an option used by the Haar wavelet transform routines. If set to True, the Haar filter is used to computed the Haar wavelet transform, else it uses {1/2,1/2} as the lowpass filter.
  • RightHWT applies the wavelet transform to each column of a. The result is a block 1x2 matrix with averages of rows in the left block and differences of rows in the right block.
  • A second option for Computation is Symbolic.
  • Symbolic computation will be overridden if either the input filter or the data are given in numerical form.
  • With Orthogonal set to False (default), the RightHWT routine uses the filter {1/2,1/2}. If Orthogonal is set to True, the routines uses the Haar filter.
  • RightHWT[data] is equivalent to RightWT[data,{1/2,1/2}] and [data,Orthogonal->False] is equivalent to RightWT[data,Haar[]];

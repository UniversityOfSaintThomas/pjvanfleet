Orthogonal


is an option used by the Haar wavelet transform routines. If set to True, the Haar filter is used to computed the Haar wavelet transform, else it uses {1/2,1/2} as the lowpass filter.

DetailsDetails

  • While Orthogonal is not an option for WPT, WPTFull and InverseWPT, it is possible to pass either the orthgononal Haar filter or {1/2,1/2} to these routines and acheive the same results.

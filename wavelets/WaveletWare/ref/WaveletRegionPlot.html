WaveletRegionPlot


or takes a vector, a matrix or list of three matrices with equal dimensions, a list of ordered pairs (vector), triples (grayscale matrix or wavelet transform of a color image) or 4-tuples (wavelet packet transform of a color image) and an optional fixed tree (packets) and returns that portion of the wavelet transform.

DetailsDetails

  • The following options can be given:
  • NumIterations1NumIterations is an option used by several functions in the WaveletWare package. It indicates the number of iterations of the wavelet transform performed on the data.
    ChannelColorWhiteChannelColor is an option for ImagePlot, WaveletPlot, FullWaveletPlot and WaveletRegionPlot. It can be used to set ColorFunction to a particular RGB color when a single matrix (with Scaling set to Image) is input in ImagePlot.
    BorderTrueBorder is an option for ImagePlot,WaveletPlot,FullWaveletPlot and WaveletRegionPlot. If set to True (default), a border is placed on the image.
    BorderColorBlackBorderColor is an option for ImagePlot, WaveletPlot, FullWaveletPlot and WaveletRegionPlot. It sets the color of the image border.
    ScalingTrueScaling is an option for ImagePlot, WaveletPlot, FullWaveletPlot and WaveletRegionPlot. If set to True (default), the data are scaled before plotting. If set to False, raw data are plotted.
    PlotTogetherFalsePlotTogether is an option for ImagePlot, WaveletPlot, FullWaveletPlot, and WaveletRegionPlot. If set to True, an input list of three matrices are plotted as a color image. Otherwise each channel is plotted separately.
    Colors{,,,,,,,,,,,,,,,,,,,,,,,,,,,,,}
    		Colors is an option for , WaveletVectorPlot and FullWaveletVectorPlot. It is a list of default colors for the iterations to be plotted.
    BackgroundWhiteBackground is an option that specifies what background color to use.
    JoinedFalseJoined is an option for ListPlot and related functions that specifies whether points in each dataset should be joined into a line, or should be plotted as separate points.
    PointSize0.01PointSize[d] is a graphics directive which specifies that points which follow are to be shown if possible as circular regions with diameter d. The diameter d is given as a fraction of the total width of the plot.
    TicksAutomaticTicks is an option for graphics functions that specifies tick marks for axes.
    ImageSizeAutomaticImageSize is an option that specifies the overall size of an image to display for an object.
    PresentationStyle{}PresentationStyle is an option for FullWaveletVectorPlot, FullWaveletPlot and that indicates how the plots should be displayed.
  • The Mathematica command that generates the list of colors for the Colors option is

        Join[ColorData[85,"ColorList"],ColorData[84,"ColorList"],ColorData[84,"ColorList"]]
  • The options NumIterations, ChannelColor, Border, BorderColor, Scaling, PlotTogether, Colors and PresentationStyle are part of the WaveletWare package. The remaining options are associated with Mathematica functions ListPlot (Background, Joined, PointSize, Ticks, ImageSize) and Image (ImageSize).
  • The input data a must be a vector, a matrix or a list of three matrices of equal dimensions.
  • If tree is not given, WaveletRegionPlot assumes that a is a (bi)orthogonal wavelet transformation. In the case where tree is given, it must be a packet tree for one- or two-dimensional data in the case were a is a vector or matrix, respectively. If the input data a is a list of three matrices then tree is either a single tree (appropriate for a matrix) or a list of three trees of the same dimension.
  • iter is a list of ordered pairs in the case that a is one-dimensional data. If a is a matrix, then iter is a list of ordered triples. If tree is a list of three trees, then iter should be a list of 4-tuples. If tree is a single tree, then iter should be a list of ordered triples.
  • The value of iter can be set to All. In this case, all regions of the wavelet (packet) transform are returned.
  • If a is one-dimensional input, then the elements of iter are of the form , where i is the iteration number and j is the region of the transform. If a is matrix or a is a list of three matrices and only one tree is given, then the elements of iter are of the form where i is the iteration, and is the block of the desired region of the transformation. If a is a list of three matrices, then the elements of iter are of the form where c is the channel 1,2,3, i is the iteration and is the block of the desired transformation.
  • In the case when a is a list of matrices and tree is a list of three trees, an element of iter can be {All,i,j,k}. In this case, the regions are extracted where c=1,2,3.
  • When a value is given for tree, the option NumIterations need not be set. The number of iterations is read from the tree structure.
  • If any of the options DivideLinesColor, ChannelColor or BorderColor are set to White, then that part of the image is plotted in gray scale.

AudioRead


is a function that reads a Mathematica audio file returning the raw data and the sample rate.

DetailsDetails

  • The following options can be given:
  • DisplayInfoFalseDisplayInfo is an option that toggles the display of image information in ImageInfo or ImageRead, audio information in AudioInfo or filter information in SplineFilters.
    PowersOfTwo0PowersOfTwo is an option that can be set in ImageRead or AudioRead so that the dimensions of the imported data is divisible by the given power of 2.

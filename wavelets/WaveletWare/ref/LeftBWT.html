LeftBWT


computes a, where is a discrete biorthogonal wavelet transform matrix and a is a matrix or a list of three matrices each of the same dimensions. The row dimension must be even.

DetailsDetails

  • The following options can be given:
  • ComputationNumericalComputation is an option for wavelet transform functions or the CE function that indicates whether the comptuation should be done numerically or symbolically.
    BoundaryFalseBoundary is an option for several routines that either compute or use the biorthogonal wavelet transform. If set to True, the symmetric biorthogonal transformation is computed.
  • LeftBWT applies the biorthogonal wavelet transform to each column of data. The result is a block 1x2 matrix with averages of columns in the top block and differences of columns in the bottom block.
  • A second option for Computation is Symbolic.
  • Symbolic computation will be overridden if either the input filter or the data are given in numerical form.
  • Boundary can be set to True to exploit the symmetric nature of a biorthogonal filter pair and provide a more accurate representation of the transformed data at the beginning and end of data sets.

AudioInfo

AudioInfo[opts]
returns the absolute path name for each audio file in the WaveletWare package. It can also supply information about the files.

DetailsDetails

  • The following options can be given:
  • DisplayInfoFalseDisplayInfo is an option that toggles the display of image information in ImageInfo or ImageRead, data information in DataInfo, audio information in or filter information in SplineFilters.
  • Audio files or folders of audio files can be added to the WaveletWare/Sounds file. Once these files are in place, they will be recognized by AudioInfo.

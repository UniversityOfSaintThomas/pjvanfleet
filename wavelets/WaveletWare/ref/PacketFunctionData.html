PacketFunctionData


or takes a lowpass filter (pair) and a nonnegative integer n and creates the nth wavelet packet function.

DetailsDetails

  • The following options can be given:
  • NumIterations10NumIterations is an option used by several functions in the WaveletWare package. It indicates the number of iterations of the wavelet transform performed on the data.
    Resolution50Resolution is an option for the ScalingFunctionData, WaveletFunctionData and to indicate the number of sample points to use per integer length of the support.
    FilterStart0FilterStart is an option for ScalingFunctionData, WaveletFunctionData and to identify the starting index of the input filter.
  • NumIterations for this function is the number of times the Cascade Algorithm iterates. The value must be an integer ⩾ 1.
  • The value for Resolution represents the number of samples to be computed per unit step of the support of the scaling function. For example, the Daubechies scaling function constructed from the length 4 orthogonal filter is supported on the interval [0,3]. Thus the default Resolution value means that 3x50=150 samples of the function will be approximated.
  • The value for FilterStart depends on the lowpass filter start index.

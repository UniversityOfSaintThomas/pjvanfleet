CheckFilter

CheckFilter[f]
takes an input and determines whether it is an orthogonal filter (1), a biorthogonal filter (2) or not a filter (-1).

DetailsDetails

  • CheckFilter checks the length of the filter(s) and also checks the (bi)orthogonality conditions.
  • Filters that are orthogonal save for the fact that they do not square sum to 1 (like {1/2,1/2}) will pass successfully through CheckFilter.
  • CheckFilter returns -1 if the input is not a (near) orthogonal filter or a biorthogonal filter pair.

Computation


is an option used by wavelet transform functions, their inverses, de-noising functions and the and cumulative energy. It indicates whether the comptuation should be done numerically or symbolically.

DetailsDetails

  • Computation can be set to either Numerical (default) or Symbolic.
  • In the case where Computation is set to Symbolic, the filter must also be in symbolic form. If this is not the case, Computation is set to Numerical.

RightLWT


computes a, where is a discrete wavelet transform matrix constructed from the LeGall filter pair and a is a matrix or a list of three matrices each with the same dimensions. The row dimension must be even.

DetailsDetails

  • The following options can be given:
  • IntegerMapFalseIntegerMapis an option used by LeGall wavelet transform functions. If set to True (default is False) and the input are integers, then the output is integer-valued.
    ComputationNumericalComputation is an option for wavelet transform functions or the CE function that indicates whether the comptuation should be done numerically or symbolically.
  • RightLWT applies the LeGall wavelet transform to each column of a. The result is a block 1x2 matrix with averages of columns in the top block and differences of columns in the bottom block.
  • A second option for Computation is Symbolic.
  • Symbolic computation will be overridden if either the input filter or the data are given in numerical form.
  • IntegerMap is a flag that ensures that RightLWT will map integer-valued input to integer-valued output.
  • If {h,ht} = Reverse[LeGall[]], then RightLWT[data] is equivalent to RightBWT[data,{-h,ht},BoundaryTrue].

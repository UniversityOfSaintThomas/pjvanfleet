HuffmanTree


takes the Huffman codes generated by MakeHuffmanCodes (first argument) and makes a Huffman tree from them.

DetailsDetails

  • The following options can be given:
  • NodeColorRGBColor[0.666667, 0.666667, 0.666667]NodeColor is an option for HuffmanTree that gives the background color for the nodes.
    NodeEdgeColorGrayLevel[0]NodeEdgeColor is an option for HuffmanTree that gives the edge color for the nodes.
    NodeEdgeThicknessMediumNodeEdgeThickness is an option for HuffmanTree that sets the thickness of the edge of the nodes.
    NodeSize0.2NodeSize is an option for HuffmanTree that sets the radius of the nodes in the tree.
    NodeFontSize12NodeFontSize is an option for HuffmanTree that sets the font size when labeling nodes in the tree.
    NodeFontColorRGBColor[0.333333, 0.333333, 0.333333]NodeFontColor is an option for HuffmanTree that sets the font color when labeling nodes in the tree.
    BranchLength0.1BranchLength is an option for HuffmanTree that sets the lengths between nodes.
    BranchColorGrayLevel[0]BranchColor is an option for HuffmanTree that sets the color of the lines between nodes.
    BranchFontSize12BranchFontSize is an option for HuffmanTree that sets the font size when labeling edges.
    BranchFontColorGrayLevel[0]BranchFontColor is an option for HuffmanTree that sets the font color when labeling edges.
    BranchThicknessMediumBranchThickness is an option for HuffmanTree that sets the thickness of the branches.
    ShowStringShowString is an option for HuffmanTree that indicates whether or not string input should be displayed.
    ImageSizeAutomaticImageSize is an option that specifies the overall size of an image to display for an object.
  • The input codes is the first element (of three) returned by MakeHuffmanCodes. It is a list of triples of the form {v,f,b} where v is the value (a character or integer), f is its relative frequency within x and b is a list of 0s and 1s that denote the binary representation of v with respect to this Huffman coding of x.
  • HuffmanTree is not designed to plot Huffman trees associated with large data sets. It is meant to be used as an instructional tool that is best utilized with a maximum number of ≈20 distinct characters.

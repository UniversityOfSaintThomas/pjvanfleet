CheckParity

CheckParity[n]
determines if a list of n positive integers are all odd (returns 1), all even (returns 2), or neither (returns 0).

DetailsDetails

  • CheckParity is a utility function that is used primarily when checking the integrity of biorthogonal filter pairs for use in wavelet transform routines.

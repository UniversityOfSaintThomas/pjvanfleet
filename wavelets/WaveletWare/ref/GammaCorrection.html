GammaCorrection


takes a grayscale image matrix a and a positive exponent r and returns an r-gamma corrected matrix.

DetailsDetails

  • The input must be a matrix whose entries are integers in the set {0,1,...,255}.
  • The formula for computing gamma correction is Round[255((a/255)^r)].

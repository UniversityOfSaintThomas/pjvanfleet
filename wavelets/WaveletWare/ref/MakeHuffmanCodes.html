MakeHuffmanCodes

MakeHuffmanCodes[x]
takes a vector, matrix or string and returns the Huffman codes for the input element as well as information about the old and new bitstream lengths.

DetailsDetails

  • If x is not a string, then x must be integer-valued. If Min[x]<0, then the data should be shifted so that Min[x] ≥ 0.
  • Output from MakeHuffmanCodes consists of a list of three elements. The first element is a list of triples of the form {v,f,b} where v is the value (a character or integer), f is its relative frequency within x and b is a list of 0s and 1s that denote the binary representation of v with respect to this Huffman coding of x.

    The second element in the output is the size (in bits) of the original input x and the last element in the output is size (in bits) of the Huffman-coded version of x.

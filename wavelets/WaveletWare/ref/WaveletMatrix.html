WaveletMatrix


takes a filter (or biorthogonal filter pair) and generates a wavelet matrix of prescribed dimensions.

DetailsDetails

  • The following options can be given:
  • Shift0Shift is an option for discrete wavelet transform functions to indicate if the lowpass/highpass filters should be shifted for the computation.
    MatrixBlockAllMatrixBlock is an option for that allows the user to determine the part (Lowpass, Highpass, All) of the matrix to construct.

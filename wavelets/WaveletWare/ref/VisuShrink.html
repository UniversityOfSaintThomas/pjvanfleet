VisuShrink

VisuShrink[data,filter,opts]
takes a noisy vector or matrix and an orthogonal filter or biorthogonal filter pair and returns a denoised approximation of the input.

DetailsDetails

  • The following options can be given:
  • NumIterations1NumIterations is an option used by several functions in the WaveletWare package. It indicates the number of iterations of the wavelet transform performed on the data.
    ThresholdByLevelFalseThresholdByLevel is a flag that tells and SUREShrink to use threshold values for each individual iteration of the wavelet transform instead of the default single value.
    ComputationNumericalComputation is an option for wavelet transform functions or the CE function that indicates whether the comptuation should be done numerically or symbolically.
    Shift0Shift is an option for discrete wavelet transform functions to indicate if the lowpass/highpass filters should be shifted for the computation.
    BoundaryFalseBoundary is an option for several routines that either compute or use the biorthogonal wavelet transform. If set to True, a symmetric biorthogonal transformation is computed.
    IntegerMapFalseIntegerMap is an option used by LeGall wavelet transform functions. If set to True (default is False) and the input are integers, then the output is integer-valued.
  • All options (except ThresholdByLevel) are for (bi)orthogonal wavelet transforms.

  • The algorithm for Visu Shrink is as follows:

    Compute k iterations of the (bi)wavelet transform on the input data;
    Estimate the noise level and use it to compute the universal threshold λuniv;
    Apply the soft threshold function ShrinkageFunction to each highpass portion of the transform;
    Join the new highpass portions of the transform with the lowpass portion;
    Apply the inverse wavelet transform

Resize


is an option for ImageRead that can be used to resize the input image.

DetailsDetails

  • The default value of Resize is False.
  • The value of Resize is sent to the Mathematica function ImageResize inside of ImageRead.
  • To remain consistent with other WaveletWare routines, in the case where Resize is a list of two integers, the values are {rows,cols}. This is in reverse of ImageResize which expects columns first.
  • If one value is given for Resize, it is assumed to be the columns for resizing - the new value of rows is determined from the Resize value and the dimensions of the original image.

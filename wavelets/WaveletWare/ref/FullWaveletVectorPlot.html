FullWaveletVectorPlot

FullWaveletVectorPlot[data,opts]
creates a list of plots of the data generated by WTFull or WPTFull.

DetailsDetails

  • The following options can be given:
  • PacketPlotFalsePacketPlot is an option for FullWaveletVectorPlot or FullWaveletPlot that indicates whether or not the input data was generated by a wavelet packet transform.
    IterationAllIteration is an option for WaveletPlot, WaveletVectorPlot, FullWaveletPlot, FullWaveletVectorPlot and WaveletVectorPlay that can be used to designate as output a particular region of the input data.
    DivideLinesTrueDivideLines is an option for WaveletPlot, FullWaveletPlot, WaveletVectorPlot and FullWaveletVectorPlot that indicates whether or not different portions of the transform should be separated by dividing lines.
    DivideLinesColorGrayLevel[0.5]DivideLinesColor is an option for WaveletPlot, FullWaveletPlot, WaveletVectorPlot and FullWaveletVectorPlot that indicates the color of divide lines.
    Colors{,,,,,,,,,,,,,,,,,,,,,,,,,,,,,}Colors is an option for WaveletRegionPlot, WaveletVectorPlot and FullWaveletVectorPlot. It is a list of default colors for the iterations to be plotted.
    ImageSizeAutomaticImageSize is an option that specifies the overall size of an image to display for an object.
    DimmingGrayLevel[0.8]Dimming is an option for WaveletPlot and WaveletVectorPlot. It sets the brightness of those portions of the transform not to be plotted.
    BackgroundWhiteBackground is an option that specifies what background color to use.
    JoinedFalseJoined is an option for ListPlot and related functions that specifies whether points in each dataset should be joined into a line, or should be plotted as separate points.
    PointSize0.01PointSize[d] is a graphics directive which specifies that points which follow are to be shown if possible as circular regions with diameter d. The diameter d is given as a fraction of the total width of the plot.
    TicksAutomaticTicks is an option for graphics functions that specifies tick marks for axes.
    PresentationStyle{}PresentationStyle is an option for , FullWaveletPlot and WaveletRegionPlot that indicates how the plots should be displayed.
  • The Mathematica command that generates the list of colors for the Colors option is

        Join[ColorData[85,"ColorList"],ColorData[84,"ColorList"],ColorData[84,"ColorList"]]
  • The options PacketPlot, NumIterations, Iteration, DivideLines, DivideLinesColor, Colors, Dimming and PresentationStyle are part of the WaveletWare package. The remaining options (ImageSize, Background, Joined, PointSize and Ticks) are associated with the Mathematica function ListPlot.
  • The input data v should be a list of vectors created by WTFull or WPTFull.
  • The value of iter can be set to All or {}. In these cases, all or none of regions of the wavelet (packet) transform are plotted.
  • The elements of iter are of the form , where i is the iteration level number 0,...,NumIterations, j is the iteration within i, and j is the region of iteration i to be plotted. For example, if FullWaveletVectorPlot is called with NumIterations set to 2, then the value {{2,1,2}} for Iteration, selects the 3rd list from v and then the highpass part (j=2) of the first iteration (i=1) of that transform.
  • FullWaveletVectorPlot calls WaveletVectorPlot n times (where n is the length of v) times to create the requested plots.

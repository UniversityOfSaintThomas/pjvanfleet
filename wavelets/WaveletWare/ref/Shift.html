Shift


is an option for discrete wavelet transform functions to indicate if the lowpass/highpass filters should be shifted for the computation.

DetailsDetails

  • The value of Shift must be an integer. If the value, in modulus, is larger than the dimension of the input, the shift is cyclic.
  • The default value for Shift is 0.

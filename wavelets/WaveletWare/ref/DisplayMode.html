DisplayMode


is an option for RGBToYCbCr, YCbCrToRGB, RGBToHSI and HSIToRGB that indicates whether or not the returned values are in the range 0,...,255.

DetailsDetails

  • DisplayMode set to True maps the output of RGBToYCbCr and RGBToHSI to a range that can be viewed by ImagePlot.
  • RGBToYCbCr takes as input integers in the range 0,1,...,255 and returns values y ∈ [0,1] and cb, cr ∈ [,]. If DisplayMode is set to True (default is False), then y ∈ [16, 235] and cb,cr ∈ [16,240]. If DisplayMode is set to True as an option for YCbCrToRGB, then the routine expects values for y ∈ [16, 235] and cb, cr ∈ [16, 240].

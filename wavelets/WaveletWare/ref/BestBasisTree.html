BestBasisTree


takes data generated by WPTFull returns the tree for the best basis to be used with the wavelet packet transform.

DetailsDetails

  • The following options can be given:
  • SecondParameterNoneSecondParameter is an option for and WPT that allows a user to input a cost function that has a second parameter.
    CostFunctionNumberNonzeroCostFunction is an option that allows the cost function to be set to something different than NumberNonzero (default) in and WPT.
  • BestBasisTree, in conjunction with WPTFull, are the primary tools used by WPT to compute the wavelet packet transform of a given vector, matrix or list of matrices.

WaveletMatrixToList


takes a wavelet-transformed matrix and partitions it into a list that is of the form lowpass, list of nth highpass triple, ..., list of first highpass triple.

DetailsDetails

  • The following options can be given:
  • NumIterations1NumIterations is an option used by several functions in the WaveletWare package. It indicates the number of iterations of the wavelet transform performed on the data.
  • WaveletMatrixToList is used primarily as a utility function for computing (bi)orthogonal wavelet transforms, but can be used to easily extract portions of the wavelet transform.

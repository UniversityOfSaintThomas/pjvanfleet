HSIToRGB


is a function that converts either an h,s,i triple or a list of h,s,i matrices to RGB space.

DetailsDetails

  • The following options can be given:
  • DisplayModeFalseDisplayMode is an option for RGBToYCbCr, YCbCrToRGB, RGBToHSI and HSIToRGB that indicates whether or not the returned values are in the range 0,...,255.
  • The range of the values in h,s,i are expected to be in the interval [0,1].
  • The function returns an r,g,b ordered triple or list of matrices. These values are in the interval [0,255].
  • The conversion algorithm is as follows. First h is mapped to 360*h (conversion to degrees).
    For < h ⩽ ,
        b=(1-s)/3
        r=(1 + )/3
        g=1-(r+b).

    For < h ≤ ,
        h=h-
        r=(1-s)/3
        g=(1 + )/3
        b=1-(r+g).

    For < h ≤
        h=h-
        g=(1-s)/3
        b=(1 + )/3
        r=1-(g+b)

    Finally, multiply each of r,g,b by 3i and round.
  • If DisplayMode is set to True, then the routine expects the h,s,i values to be in the interval [0,255].

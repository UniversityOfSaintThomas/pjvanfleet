WaveletTree


takes a number of iterations and generates the basis tree for a standard wavelet transform.

DetailsDetails

  • The following options can be given:
  • InputDimension1InputDimension is an option for that instructs the routine to make a tree for one-dimensional (default) or two-dimensional data.
  • InputDimension should either be set to 1 (default) for trees for packet transforms of vectors or 2 for packet transforms of (lists of) matrices.
  • WaveletTree can be used in conjunction with WPT to compute (biorthogonal) wavelet transforms. For example, if v is a vector, then the command WPT[v,Daub[4],NumIterations->2,FixedTree->WaveletTree[2]] is equivalent to WT[v,Daub[4],NumIterations->2].

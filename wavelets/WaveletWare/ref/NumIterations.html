NumIterations


is an option used by several functions in the WaveletWare package. It indicates the number of iterations of the wavelet transform performed on the data.

DetailsDetails

  • NumIterations must be a nonnegative integer. The default value for NumIterations in transform and plotting routines is 1.
  • NumIterations is used in a different capacity for WaveletFunctionData, ScalingFunctionData and PacketFunctionData. Here it indicates the number of cascade algorithm steps to perform. The default value of NumIterations for these three functions is 10.

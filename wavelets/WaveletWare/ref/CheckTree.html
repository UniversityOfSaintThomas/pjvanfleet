CheckTree


determines, given datatype, if an input tree is valid for use with a packet transform.

DetailsDetails

  • CheckTree is a utility function used primarily by packet transform modules and wavelet plotting modules to make sure the input tree(s) are of the correct form.
  • CheckTree, if successful, returns the number of iterations of the transform as indicated by the length of the tree.

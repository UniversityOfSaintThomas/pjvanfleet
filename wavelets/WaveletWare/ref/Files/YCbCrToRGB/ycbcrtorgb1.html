When DisplayMode is set to True, the values sent to YCbCrToRGB are y ∈ [16,235] and cb,cr ∈ [16,240]. These values are obtained by setting DisplayMode to True in RGBToYCbCr. These values are converted to y',cb',cr' using the affine maps

y' = (y-16)/219
cb'= (cb-128)/224
cr' = (cr-128)/224

and then the equations described above are used to convert y',cb',cr' to r,g,b values that are integers in the set {0,1,...,255}.

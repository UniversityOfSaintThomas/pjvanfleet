Iteration  (2)Iteration  (2)

By default, WaveletVectorPlot plots all iterations of the wavelet transform, but the option Iteration can be used to highlight particular iteration. In this case, Iteration should be given as a list of ordered pairs where i is the iteration and j is the portion of the ith iteration. Note that only in the case when i equals NumIterations, j=1,2. For i<NumIterations, j=2.

Regions not identified by Iteration are plotted in a light gray color. The grav scale value of the regions not selected can be controlled by the Dimming option.

The default value of Iteration is All.

Colors  (4)Colors  (4)

WaveletVectorPlot plots the different regions of the wavelet transform using the default colors given by the Colors options. The length of the Colors list is 30. If the transform consists of more than 30 regions, extra colors from the beginning of the list are added. The user can control the colors of each plotted region.

Note that the length of Colors needs to be at least as long as NumIterations + 1.

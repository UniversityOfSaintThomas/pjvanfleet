ThresholdByLevel  (1)ThresholdByLevel  (1)

The VISUShrink algorithm computes the universal threshold by using the highpass portion of the first iteration of the wavelet transform. This is the case when ThresholdByLevel is set to its default value of False. Setting ThresholdByLevel to True instructs VISUShrink to compute a universal threshold for each highpass portion of the wavelet transform and use it with ShrinkageFunction to quantize the highpass portions of the transform.

Computation  (1)Computation  (1)

By default, all computations of the wavelet transform are numerical. However, if input data are symbolic, the filter values are symbolic and Computation is set to Symbolic, then the computations are done symbolically. Note that if the filter is given in numerical format, then the computations are done numerically.

Create integer-valued data and integer-valued noise so that the wavelet transform can be computed symbolically.

Shift  (1)Shift  (1)

The default value for Shift is 0. This means the orthogonal filter processes the data starting with the first entry and continues, moving over 2 each time. Setting Shift to a different integer changes the starting point. This option can be set for the wavelet transform used in VISUShrink.

Boundary  (1)Boundary  (1)

For symmetric biorthogonal filter pairs, Boundary can be set to True and in this case, BWT exploits the symmetry of the filters to more accurately represent the transform at the beginning and end of the data set.

In the case where the filter lengths are both odd, BWT uses as input the (2n-2)-length vector

        (, ..., , , , ..., , )

and in cases where the filter lengths are both even, BWT uses as input the 2n-length vector

        (, ..., , , , ,..., )

and in both cases, takes the first n/2 values of the output for the lowpass portion and the first n/2 values of the last n-1 (filter lengths odd) or n (filter lengths even) components for the highpass portion to build the transform.

VISUShrink can use biorthogonal filters and exploit the Boundary setting.

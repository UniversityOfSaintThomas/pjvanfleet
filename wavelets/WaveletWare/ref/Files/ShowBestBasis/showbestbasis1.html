OpacityOn  (1)OpacityOn  (1)

For the color of the regions represented by the best basis tree of a 2-dimensional packet transform with Show3D set to True, OpacityOn sets the Opacity of the color. The default value is 0.75. Any value in the interval [0,1] can be entered. Opacity set to 0 gives perfect transparency.

NumIterations  (1)NumIterations  (1)

By default, wavelet transform functions only compute one iteration of the wavelet transform. If the dimensions of the data are divisible by , then up to p iterations of the transform can be computed. If p iterations of the wavelet transform have been computed, then setting NumIterations to p will play the correct portions of the transform.

Iteration  (1)Iteration  (1)

The Iteration options allows the user to control which portions of the wavelet (packet) tranform is played. The default value for Iteration is All, in which case all parts of the transform are played. The value can instead be a list of ordered pairs {i,j}, where i indicates the iteration and j indicates the portion of the iteration (j=1 is lowpass, j=2 is highpass) to be played. Note that for wavelet transforms, j=1 only when i=NumIterations.

Play the highpass portions of a wavelet transform.

PlotTogether set to True may not work if the elements in Iteration are not properly ordered. For lists of three matrices to be plotted together, WaveletPlot partitions the values for Iteration into groups of three and then checks that each group is of the form {{1,i,j,k},{2,i,j,k},{3,i,j,k}}. An error message might appear even though the elements of Iteration are correct. In such a case, a simple reordering of the elements in Iteration should work.

Iteration  (5)Iteration  (5)

By default, FullWaveletPlot renders every region for all iterations of the wavelet transform. The Iteration option can be used to control which regions are plotted. If a value other than All is given for Iteration, then it must be a list of length n (where n is the number of matrices or lists of matrices from the input a) and each element in the list can be All, {} (no regions plotted), or a list of regions to plot. The list of regions to plot must be valid regions. If a is a matrix, these regions are denoted by triples {i,j,k} where i is the iteration and (j,k) is the block when the matrix is partitioned in blocks of size (rows,cols)/. In the case where each element of a is a list of three matrices, then elements in the list for iter should have elements of the form {c,i,j,k} where i,j,k are as described previously and c is the channel, c=1,2,3. The element {All,i,j,k} can be used in place of {1,i,j,k}, {2,i,j,k}, {3,i,j,k}.

DimmingRange  (4)DimmingRange  (4)

DimmingRange is the value assigned to IntensityRange in the MatrixStretch function. All dimmed portions of the transform are scaled by default. The default value of DimmingRange is {200,255} which scales the region in a lighter tone of the dimming color. DimmingRange must be of the form {a,b}, where a ⩽ b are integers in the range 0,...,255. Alternatively, DimmingRange can be set individually for each iteration of the plot. In this case DimmingRange is a list of ordered pairs {a,b} of length n, where n is the length of input a.

Scaling  (1)Scaling  (1)

If Scaling is set to True (the default is False), then MatrixStretch is applied to the input before the image is displayed. The result is the minimum/maximum absolute values of the image (or channel) being mapped to 0/255 and intermediate absolute values then mapped linearly to a value 1,...,254.

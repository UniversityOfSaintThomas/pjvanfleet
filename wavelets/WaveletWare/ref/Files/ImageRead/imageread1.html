Resize  (1)Resize  (1)

ImageRead can Resize an image using Mathematica's ImageResize function. The default value of Resize is False which means no resizing occurs. The value can be a single positive integer c, in which case the image is resized to have c columns and a proportional number of rows. If Resize is set to a list of 2 positive integers then the resized image has r rows and c columns.

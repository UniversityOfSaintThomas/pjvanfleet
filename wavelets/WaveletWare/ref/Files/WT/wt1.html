Shift  (1)Shift  (1)

The default value for Shift is 0. This means the orthogonal filter processes the data starting with the first entry and continues, moving over by 2 each time. Setting the shift to a different integer changes the starting point.

Computation  (1)Computation  (1)

By default, all computations are numerical. However, if the input data are symbolic, the filter values are symbolic and Computation is set to Symbolic, then the computations are done symbolically. Note that if the filter is given in numerical format, then the computations are done numerically.

If the length of the list of regions to be plotted is 2, then the ImageSize option might not work correctly. This is due to the nature of the way ImageSize can accept values. The best is to give a list of length two with each element being a list of length two (the row and column size of the image).

If the length of the list of regions to be plotted is 2, then the Ticks option might not work correctly. This is due to the nature of the way Ticks can accept values. The best is to give explicit ranges for each region - that is, create a list of length two but then each of these lists is a list of two elements that are the ticks for the horizontal and vertical axes, respectively.

When plotting color images, ChannelColor set to a list of three colors will trump PlotTogether set to True. When ChannelColor is set to a list of three colors, the channels are automatically plotted separately.

If PlotTogether is set to True, then all data are plotted in color - WaveletRegionPlot does not allow some regions to be plotted together and others not.

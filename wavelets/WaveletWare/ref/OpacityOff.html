OpacityOff


is an option for ShowBestBasis that sets the opacity level for the elements that are not part of the best basis.

Use different levels of opacity for the regions not in the best basis tree. Show3D must be set to True. OpacityOff is a value in the interval [0,1] with the default value 0.3. Set OpacityOff is set to 0 to get perfect transparency.

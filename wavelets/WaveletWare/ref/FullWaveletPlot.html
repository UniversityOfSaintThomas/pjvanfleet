FullWaveletPlot


creates a list of plots of the data generated by WTFull or WPTFull.

DetailsDetails

  • The following options can be given:
  • PacketPlotFalsePacketPlot is an option for FullWaveletVectorPlot or that indicates whether or not the input data was generated by a wavelet packet transform.
    IterationAllIteration is an option for WaveletPlot, WaveletVectorPlot, FullWaveletPlot, FullWaveletVectorPlot and WaveletVectorPlay that can be used to designate as output a particular region of the input data.
    PlotTogetherFalsePlotTogether is an option for ImagePlot, WaveletPlot, FullWaveletPlot, and WaveletRegionPlot. If set to True, an input list of three matrices are plotted as a color image. Otherwise each channel is plotted separately.
    ScalingTrueScaling is an option for ImagePlot, WaveletPlot, FullWaveletPlot and WaveletRegionPlot. If set to True (default), the data are scaled before plotting. If set to False, raw data are plotted.
    DimmingGrayLevel[0.8]Dimming is an option for WaveletPlot and WaveletVectorPlot. It sets the color of those portions of the transform not to be plotted.
    DimmingRange{200, 255}DimmingRange is an option for WaveletPlot or FullWaveletPlot. It sets the intensity range for those portions of the wavelet transformed that are dimmed.
    DivideLinesTrueDivideLines is an option for WaveletPlot, FullWaveletPlot, WaveletVectorPlot and FullWaveletVectorPlot that indicates whether or not different portions of the transform should be separated by dividing lines.
    DivideLinesColorWhiteDivideLinesColor is an option for WaveletPlot, FullWaveletPlot, WaveletVectorPlot and FullWaveletVectorPlot that indicates the color of divide lines.
    BorderTrueBorder is an option for ImagePlot,WaveletPlot,FullWaveletPlot and WaveletRegionPlot. If set to True (default), a border is placed on the image.
    BorderColorBlackBorderColor is an option for ImagePlot, WaveletPlot, FullWaveletPlot and WaveletRegionPlot. It sets the color of the image border.
    ChannelColorWhiteChannelColor is an option for ImagePlot, WaveletPlot, FullWaveletPlot and WaveletRegionPlot. It can be used to set ColorFunction to a particular RGB color when a single matrix (with Scaling set to Image) is input in ImagePlot.
    PresentationStyle{}PresentationStyle is an option for FullWaveletVectorPlot, FullWaveletPlot and WaveletRegionPlot that indicates how the plots should be displayed.
    ColorFunctionAutomaticColorFunction is an option for graphics functions that specifies a function to apply to determine colors of elements.
    ColorFunctionScalingTrueColorFunctionScaling is an option for graphics functions that specifies whether arguments supplied to a color function should be scaled to lie between 0 and 1.
    AlignmentPointCenterAlignmentPoint is an option which specifies how objects should by default be aligned when they appear in Inset.
    BaselinePositionAutomaticBaselinePosition is an option that specifies where the baseline of an object is considered to be for purposes of alignment with surrounding text or other expressions.
    ColorSpaceAutomaticColorSpace is an option for Image and related functions that specifies the color space to which color values refer.
    ImageResolutionAutomaticImageResolution is an option for Export, Rasterize, and related functions that specifies at what resolution bitmap images should be rendered.
    ImageSizeAutomaticImageSize is an option that specifies the overall size of an image to display for an object.
    InterleavingAutomaticInterleaving is an option for Image and related functions that specifies whether data corresponding to different channels in an object such as an image should be interleaved.
    MagnificationAutomaticMagnification is an option for Style and Cell that specifies what magnification to use for display.
    MetaInformation{}MetaInformation is an option giving meta-information for Image, CloudObject, and other objects.
    TaggingRules{}TaggingRules is an option for selections that specifies a list of strings to be associated with a selection.
  • The input a should be a list of matrices (or a list of lists of three matrices) created by WTFull or WPTFull.
  • The options PacketPlot, Iteration, PlotTogether, Scaling, Dimming, DimmingRange, DivideLines, DivideLinesColor, Border, BorderColor, ChannelColor and PresentationStyle are part of the WaveletWare package. The remaining options are associated with the Mathematica functions Image and Raster.
  • If a is a list of n matrices, then the WaveletWare options listed (except for PacketPlot and PresentationStyle) above can either be entered as a single value (in which case it will be converted to a constant list of n of these values) or a list of n values.
  • FullWaveletPlot basically parses the values from a and the given option values and makes n calls to WaveletPlot. Here n is the number of matrices in a.
  • The value(s) of iter can be set to All or {}. In these cases, all or none of regions of the wavelet (packet) transform are plotted.
  • If a is a list of matrices, then the elements of iter are of the form where i is the iteration, and is the block of the desired region of the transformation. If a is a list of lists of three matrices, then the elements of iter are of the form where c is the channel 1,2,3, i is the iteration and is the block of the desired transformation.
  • iter can be {All,i,j,k}. In this case, the regions are extracted where c=1,2,3.

CheckIterationValue


checks the validity of a given value of Iteration.

DetailsDetails

  • Iteration is a value for several wavelet plotting routines. CheckIterationValue checks the integrity of this value.
  • datatype should either be 1 if the tree was built for a vector or 2 if the tree was built for a matrix.
  • In the case where datatype is 1, iter should take the form {i,j} where i is the iteration and j is the region (j=1,2 for wavelet transforms, j=1,..., for wavelet packet transforms).
  • In the case where datatype is 2, iter should take the form {i,j,k} where i is the iteration and (j,k) is the appropriate block from iteration i (j, k=1,2 for wavelet transforms, j, k=1,..., for wavelet packet transforms).

DataInfo

DataInfo[opts]
returns the absolute path name for each tabular text or spreadsheet in the WaveletWare package. It can also supply information about the files.

DetailsDetails

  • The following options can be given:
  • DisplayInfoFalseDisplayInfo is an option that toggles the display of image information in ImageInfo or ImageRead, data information in DataInfo, audio information in AudioInfo or filter information in SplineFilters.
  • Data files or folders of data files can be added to the WaveletWare/Data file. Once these files are in place, they will be recognized by DataInfo.
  • See the tutorial listed below to learn how to add your own data files to the WaveletWare package.

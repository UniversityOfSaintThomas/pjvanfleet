CheckData

CheckData[data]
takes an input argument and determines whether it is a vector (1), matrix (2) or list of three matrices (3). The routine also returns the maximum number of iterations of a wavelet transform that can be performed on the input.

DetailsDetails

  • CheckData is a utility routine that attempts to determine if the data are formatted correctly for use in several functions in the WaveletWare package.
  • The output of CheckData is a list of two integers .

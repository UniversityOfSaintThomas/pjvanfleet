DetailsDetails

  • The following options can be given:
  • DisplayFunction$SoundDisplayFunctionDisplayFunction is an option for graphics and sound functions that specifies a function to apply to graphics and sound objects before returning them.
    Epilog{}Epilog is an option for graphics functions that gives a list of graphics primitives to be rendered after the main part of the graphics is rendered.
    PlayRangeAutomaticPlayRange is an option for Play and related functions which specifies what range of sound amplitude levels should be included.
    Prolog{}Prolog is an option for graphics functions which gives a list of graphics primitives to be rendered before the main part of the graphics is rendered.
    SampleDepth8SampleDepth is an option for sound primitives that specifies how many bits should be used to encode sound amplitude levels.
    SampleRate11025SampleRate is an option for sound primitives that specifies the number of samples per second to generate for sounds.
    NumIterations1NumIterations is an option used by several functions in the WaveletWare package. It indicates the number of iterations of the wavelet transform performed on the data.
    IterationAllIteration is an option for WaveletPlot, WaveletVectorPlot, FullWaveletPlot, FullWaveletVectorPlot and WaveletVectorPlay that can be used to designate as output a particular region of the input data.
  • The options NumIterations and Iteration are part of the WaveletWare package. The remaining options, DisplayFunction, Epilog, PlayRange, Prolog, SampleDepth and SampleRate are options for the Mathematica function ListPlay.

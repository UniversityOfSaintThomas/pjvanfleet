CompositeTree


is an option for WPT that when set to True (default is False) and the input is a list of three matrices, creates a single best basis tree.

DetailsDetails

  • In the case when the input for WPT is a list of three matrices, the routine computes a best basis tree for each matrix. However, setting CompositeTree to True instructs WPT to compute a single best basis tree based on the mean of the three input matrices.

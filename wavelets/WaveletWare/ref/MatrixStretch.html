MatrixStretch


linearly maps Abs[a] from (Min[a],x) to (Max[a],y).

DetailsDetails

  • The following options can be given:
  • IntensityRange{0, 255}IntensityRange is an option for . It is a list of two values indicating the min/max of the intensity range used to scale the input matrix.
  • MatrixStretch is a utility function internal to WaveletWare that is used by 2D plotting functions such as ImagePlot, WaveletPlot, FullWaveletPlot and WaveletRegionPlot to scale portions of the data for display.

ThresholdByLevel


is a flag that tells VisuShrink and SUREShrink to use threshold values for each individual iteration of the wavelet transform instead of the default single value.

In[1]:=

Typically, wavelet shrinkage routines compute a threshold value for the quantization step by using the highpass portion of the first iteration of the wavelet transform of the noisy data. This is the case in VisuShrink and SUREShrink when ThresholdByLevel is set to its default value False. Setting the value to True instructs the shrinkage routines to compute a new threshold value for each highpass portion of the transform.

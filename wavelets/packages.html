DiscreteWavelets Software Package

DiscreteWavelets is a collection (package/toolbox) of functions written to supplement the text Discrete Wavelet Transformations: An Elementary Approach with Applications, John J Wiley & Sons, ©2008, by Patrick J. Van Fleet.

The collection contains modules for deriving orthogonal and biorthogonal wavelet filters and using them to compute one- and two-dimensional(biorthogonal) discrete wavelet transformations. Modules are also included so that the user can visualize digital images and their transformations as well as hear digital audio files and their transformations. Other modules allow the user to perform naive image compression or de-noise signals or audio clips. These routines include cumulative energy, entropy, PSNR, DCTs, gamma correction, histogram equalization, universal shrinkage tolerance, SureShrink tolerance, wavelet shrinkage, and SureShrink de-noising. There are also modules that allow the user to investigate Fourier series and perform color space conversions.

The package is currently available for use with Mathematica (versions 5.2-8.0) and MATLAB (R2007a and later). The package works with Mathematica 9 but there are difficulties accessing the documentation. We are working with Wolfram to solve this problem.

A new version of the DiscreteWavelets package is currently under development. This package will upgrade the speed and efficiency of several functions, update the documentation, and introduce new functions that makes the package a supplement to the text Wavelet Theory: An Elementary Approach with Applications, John J Wiley & Sons, ©2009, by David K. Ruch and Patrick J. Van Fleet.

Download and installation instructions appear below:

Mathematica

  1. Click here to download DiscreteWavelets for Mathematica.
  2. Find the correct installation directory. Open Mathematica and type:
    • $BaseDirectory if you have administrator privileges on the computer and want to install the package for all users.
    • $UserBaseDirectory if you want to install the package just for your use.
  3. Unzip the zip file from Step 1 into the Applications sub-folder of the folder returned from Step 2.
  4. Common Errors:
    • The extraction produces two DiscreteWavelets folders under the Applications sub-folder.
    • The extraction doesn't make it to the Applications sub-folder.
  5. Go back to Mathematica and issue the command <<DiscreteWavelets`DiscreteWavelets` (take care to get the grave accent, not the apostrophe when typing this command)
  6. If you see no error messages after issuing the above command, chances are the installation was successful. To double-check, type the command ?Haar. If the installation worked, you should see a message that says Haar returns the Haar filter.

MATLAB

  1. Click here to download DiscreteWavelets for MATLAB.
  2. Unzip the zip file to a place on your hard drive where you want the toolbox to reside.
  3. The zip file will unzip one (large) folder called DiscreteWavelets. Important: Do not rename this folder or any subfolders inside it!! Several functions will not work if the directory structure is changed.
  4. Open MATLAB.
  5. Click on File and then Set Path...
  6. Click the Add with SubFolders... button and then navigate to where you unzipped the DiscreteWavelets folder.
  7. Click OK and then Save and Close
  8. In the command window in MATLAB, type the command rehash and then startup. The second command initializes some global variables used by the toolbox. If you received a message that global variables have been initialized, then you are ready to go!
